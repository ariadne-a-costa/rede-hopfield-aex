# Rede de Hopfield para recuperação de padrões

Implementação em Python de uma Rede de Hopfield capaz de armazenar e
recuperar representações binárias 7×7 das letras A, E e X.

O programa adiciona ruído aos padrões, realiza a recuperação assíncrona
e gera gráficos de energia e distância de Hamming.

## Tecnologias

- Python
- NumPy
- Matplotlib
